# Security Policy

## Supported versions

Only the latest `main` branch of classiflow-platform is supported. For classiflow
and taxomind, refer to their repositories.

## Reporting a vulnerability

Please report security issues in the repository where the issue exists:

- classiflow: https://github.com/rowsquared/classifAI
- taxomind: https://github.com/rowsquared/taxomind
- classiflow-platform: https://github.com/rowsquared/classiflow-platform

Use GitHub Security Advisories where available. Avoid posting sensitive details
in public issues.
