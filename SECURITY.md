# Security Policy

## Supported versions

Only the latest `main` branch of classifai-platform is supported. For classifAI
and taxomind, refer to their repositories.

## Reporting a vulnerability

Please report security issues in the repository where the issue exists:

- classifAI: https://github.com/rowsquared/classifAI
- taxomind: https://github.com/rowsquared/taxomind
- classifai-platform: https://github.com/rowsquared/classifai-platform

Use GitHub Security Advisories where available. Avoid posting sensitive details
in public issues.
